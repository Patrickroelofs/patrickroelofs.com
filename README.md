# Patrick Roelofs

A personal portfolio website to showcase my work and skills, feel free to be inspired.

## General info

This project is a personal portfolio website to showcase my work and skills. The website is build with Astro, Tailwind and Typescript.

## Technologies

Project is created with:

- Astro
- Tailwind
- Typescript

## Setup

To run this project, install it locally using npm:

```bash
pnpm install
pnpm run dev
```
